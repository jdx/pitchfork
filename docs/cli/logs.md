# `pitchfork logs`

- **Usage**: `pitchfork logs [-n <N>] [-t --tail] [ID]...`
- **Aliases**: `l`

Displays logs for daemon(s)

## Arguments

### `[ID]...`

Show only logs for the specified daemon(s)

## Flags

### `-n <N>`

Show N lines of logs

Set to 0 to show all logs

### `-t --tail`

Show logs in real-time
